# PdfCC

App sencilla para la gestion de archivos PDF, inspirada por PDFTK builder (http://www.angusj.com/pdftkb/)

## Arquetipo utilizado

- JRG Simple (https://github.com/javierrgonz/spring_archetypes_simple.git)

## Versiones

- Java: 1.8
- Spring: 5.2.5.RELEASE
- TinyLog: 2.0.0-RC1
- Lombok: 1.16.16
